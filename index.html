How to Use

1

Install Extension

Download and install the extension for your browser from the Chrome Web Store or Safari Extensions Gallery.

2

Visit Any Page

Navigate to any webpage containing Mermaid diagram code (GitHub, Confluence, Jira, documentation sites, etc.).

3

Click Extension

Click the pink Mermaid Visualizer icon in your browser toolbar to detect and visualize all diagrams.

4

Enjoy Diagrams

View beautiful, interactive diagrams with navigation, zoom, and export to mermaid.live.
